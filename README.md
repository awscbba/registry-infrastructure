# Registry Infrastructure

AWS CDK infrastructure for the People Registry system.

## Quick Start

```bash
# Install dependencies
npm install

# Deploy infrastructure
cdk deploy

# Run tests
npm test
```

## Documentation

All documentation is maintained in the `registry-documentation` repository.

## Repository Structure

- `lib/` - CDK constructs and stacks
- `lambda/` - Lambda deployment artifacts
- `test/` - Infrastructure tests

For detailed documentation, deployment guides, and architecture information, see the `registry-documentation` repository.
